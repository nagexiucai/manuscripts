# 列表 #

## 行还是列 ##
本意想做一个类似C语言中的二维数组。

- 第一种：列表解析。

> _ = [["{x}{y}".format(x=x,y=y) for x in range(5)] for y in range(5)]
> 
> _[2][2] = "just modify (2,2)"

- 第二种：算数扩张。

> __ = [[0]*5]*5
> 
> __[2][2] = None

分别执行上述两种代码片段，结果肯定吃惊！

为什么第一种和预想的一样？为什么第二种和预想的不同？
