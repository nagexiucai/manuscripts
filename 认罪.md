# 认罪——旧文 #
到了该付出代价的时候

你还想逃走

脱掉外衣裸露（lou）

看自己有多丑

满是泪水的双眸

能顶个球

不要总觉得往事不堪回首

都是自作自受

快乐悲伤哀愁

样样都有

磕头

不要继续装牛

辩驳的话免开尊口

去她妈各种理由

虔诚祈求

这不算太糗

认完罪后

动起那拙劣双手

再爬一层楼
